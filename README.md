## What is Learn-Link?

Learn-Link is a full-stack e-learning platform where students can access
courses and instructors can publish content — each with isolated, role-specific
permissions enforced via JWT and RBAC.

Built with React, Node.js, Express, and MongoDB. Features OpenAI-powered
content moderation on user submissions, 10+ REST API endpoints, and a
responsive UI with lazy loading and code splitting for fast initial load.
